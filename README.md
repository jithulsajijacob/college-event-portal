# College Event Management Portal

A web-based platform for managing college events, developed using **HTML, CSS, JavaScript, PHP, and MySQL**.  
This project is part of the _Internet and Web Programming_ course.

## Features

- User authentication (students, organizers, admin)
- Event creation, registration, and feedback
- AJAX-based dynamic registration
- Admin dashboard with analytics

## Technologies

HTML5 · CSS3 · JavaScript (jQuery) · PHP (PDO) · MySQL · AJAX · Java (JDBC)
